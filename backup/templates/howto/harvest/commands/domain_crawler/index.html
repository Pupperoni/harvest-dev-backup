Default

Using the command -d <domain_id> or --domain <domain_id>, Harvest will query Granary WS and perform site crawling.

python Harvest.py -d <domain_id>

Register Pattern

Using the command -r or --register, Harvest will check for newly created pattern in local machine and add this to Granary.

python Harvest.py --register;

Initialize Farmer

Using the command --init_farmer <name> or --init_farmer=<name>, Harvest will the run the target pattern for the 1st time and add this to Granary.

python Harvest.py --init_farmer <name>
