List all Domains

Using the command -l or --list, Harvest will query Granary WS and list available pattern in Production.

python Harvest.py --list

Get Pending pages for download.

Using the command --pending, Harvest will query Granary WS and list all pending product pages for download.

python Harvest.py --pending

Run as Client.

Using the command --client <seconds_interaval>, Harvest will run as CLIENT to perform task based on received SQS message.

python Harvest.py --client <seconds_interaval>
