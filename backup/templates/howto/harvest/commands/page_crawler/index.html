Default

Using the command -p <domain_id> or --page <domain_id>, Harvest will run page crawler pattern and download all pending product pages.

python Harvest.py -p <domain_id>

Download Once

Using the command -p <domain_id> -o or --page <domain_id> -o, Harvest will run page crawler pattern and download 1 product page then terminate.

python Harvest.py -p <domain_id> -o
