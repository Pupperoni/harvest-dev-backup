Format: def get_url(xpath=None, name=None, id=None, tag = "href", sleep=False, identifier=None)

Parameter Data Type Required Description
xpath String Yes Get the target object based on XML path.
name String Optional Get the target object based on HTML name attribute.
id String Optional Get the target object based on HTML id attribute.
tag Integer Optional Extract the url from target tag. Default target href
sleep Boolean Optional Set the Harvest on sleep prior extracting the url. Intended for sites with lots of ads, videos, etc.
identifier String Optional NewExtract the url from target object and also extract the text.

Default target innerText
Note: I'm extracting extracting all url(s) in string based on this regular expression //(?:[a-zA-Z]|[0-9]|[$-_@.&+]|[!*\(\),]|(?:%[0-9a-fA-F][0-9a-fA-F]))+
Return Data: List 

[
    {
        "url": "insert url here",
    }
]

New Extracting the version from the target Object text.

[
    {
        "url": "insert url here",
        "identifier": {
            "id": "identifier here",
        }
    }
]
